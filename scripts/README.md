# Scripts

Manual E2E / flow test scripts. Run with Node.js against a running backend (e.g. `http://localhost:3000`).

- `test-complete-flow.js` – full flow (register, login, friend request, accept, etc.)
- `test-delete-conversation-fix.js` – delete conversation flow
- `test-friend-requests.js` – friend request flow
- `test-friend-system.js` – friend system flow
- `test-friends.js` – friend-related HTTP + WebSocket tests

Not part of the shipped app; for development and manual verification.
