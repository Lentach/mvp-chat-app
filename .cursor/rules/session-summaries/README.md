# Session Summaries

Ten folder zawiera streszczenia sesji z agentem AI. Każda sesja kończy się zapisem podsumowania.

**Dla agentów:**
- **Na START sesji:** Przeczytaj `LATEST.md` (lub najnowszy plik wg daty) aby zrozumieć kontekst poprzednich prac.
- **Na KONIEC sesji:** Zapisz streszczenie w nowym pliku `YYYY-MM-DD-session.md` i zaktualizuj `LATEST.md`.

**Format plików:** `YYYY-MM-DD-session.md` (np. `2026-02-01-session.md`)
